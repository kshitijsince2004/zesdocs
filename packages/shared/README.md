# Zesdocs Shared Package

Shared TypeScript types, utilities, and components for Zesdocs.

## Features

- Common TypeScript interfaces
- Utility functions
- Shared constants
- API client types

## Usage

```typescript
import { Link, User, isValidUrl } from '@zesdocs/shared';
```

## Development

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```
