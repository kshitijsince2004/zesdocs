# Zesdocs Mobile App

React Native mobile application for Zesdocs - The Smart Link Hub.

## Features

- Quick link capture
- QR code scanning
- Voice input
- Offline sync
- Push notifications

## Development

### Prerequisites

- Node.js 18+
- React Native CLI
- Android Studio (for Android)
- Xcode (for iOS)

### Setup

```bash
pnpm install
```

### Run

```bash
# Android
pnpm run android

# iOS
pnpm run ios
```
